# 🖼️ Gallery App

A modern and responsive image gallery application that fetches beautiful photos and author information from the Lorem Picsum API. Browse through curated images with smooth pagination and a clean user interface.

## 🌐 Live Preview

Check out the application in action: **[Gallery App Live Demo](https://YugAgarwal005.github.io/Gallery-App/)**

## ✨ Key Features

* **Dynamic Image Gallery:** Fetches high-quality images directly from the Lorem Picsum API.
* **Author Information:** Displays the photographer/author name for every image.
* **Pagination Support:** Easily navigate between pages of images using Next and Previous controls.
* **Responsive Design:** Optimized for desktops, tablets, and mobile devices.
* **Fast Performance:** Built with React and Vite for a smooth browsing experience.

## 🛠️ Built With

* **ReactJS** for building dynamic, component-based user interfaces.
* **Tailwind CSS** for utility-first styling and responsive layouts.
* **Vite** for lightning-fast development and optimized production builds.
* **Lorem Picsum API** for fetching images and author information.

## 🚀 How to Use

1. Visit the [Gallery App](https://YugAgarwal005.github.io/Gallery-App/).
2. Browse the collection of images displayed on the page.
3. View the author's name below each image.
4. Click **Next** to load the next page of images.
5. Click **Prev** to return to the previous page.

## ⚙️ Getting Started Locally

Want to run or modify this project on your own machine? Follow these simple steps:

### 1. Clone the repository

```bash
git clone https://github.com/YugAgarwal005/Gallery-App.git
```

### 2. Navigate to the project directory

```bash
cd Gallery-App
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

### 5. Open in your browser

The development server will provide a local URL (typically `http://localhost:5173`) that you can open in your browser.

## 🔗 API Used

This project uses the **Lorem Picsum API** to fetch image data and author information dynamically.
